VestileFit – Outfit Recommendation System

Description

    VestiLeFit is a web-based application that generates personalized outfit recommendations based on user attributes such as body type, size, age, gender, weather, and occasion.

    It combines fit analysis with a rule-based recommendation system and provides explanations for each recommendation.


Features

- Body type detection and size recommendation
- Multi-criteria outfit filtering
- Category-based recommendations (Top, Bottom, Dress)
- Explainable results based on matched conditions
- Interactive interface for browsing outfits


Technologies Used

- Python (Flask)
- MySQL
- HTML, CSS, JavaScript


System Approach

    The system uses rule-based filtering to match user inputs with outfit attributes.
    Recommendations are generated by combining constraints such as body type, occasion, and weather, and explanations are produced based on these matches.


Limitations

- Rule-based system (no machine learning)
- Limited dataset and category granularity


Future Improvements

- Integrate machine learning for improved recommendation
- Expand dataset with detailed clothing attributes
- Improve ranking and personalization
